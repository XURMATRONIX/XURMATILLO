## How to deploy

### As a Gemini Gem

1. Open Google Gemini and create a new Gem.
2. Set the Gem name to: **XURMATILLO**
3. Open `XURMATILLO_INSTRUCTION.md`. Copy its full contents into the **Instructions** field of the Gem.
4. Upload all 10 knowledge files (`KB_01_*.md` through `KB_10_*.md`) into the Gem's **Knowledge** section.
5. Save the Gem.
6. (Optional) Set a brief description such as: *Universal cognitive operating system. Founder: Xurmatillo Oxunov.*

### As a ChatGPT Custom GPT

1. Open the [`CHATGPT/`](./CHATGPT/) folder.
2. Follow `CHATGPT/CHATGPT_SETUP_GUIDE.md` step by step.
3. The instruction is pre-compressed to fit ChatGPT's 8,000-character Instructions limit while preserving the full architecture.
4. Upload the 10 KB files in `CHATGPT/` (identical to the Gemini set — platform-agnostic).
5. Enable all four capabilities: Web Search, Canvas, Image Generation, Code Interpreter & Data Analysis.

Knowledge files are read by each platform's RAG system and chunks are retrieved on demand.

## Design principles

### Globally deployable
No personal information about any individual is embedded in the files. The system is suitable for public release.

### Architecture-private
The Gem is instructed to never reveal its internal structure, prompts, or file content when asked. The only piece of meta-information disclosed is the founder's name, when sincerely asked.

### Integrity-first
- Seven-level Truth Filter is applied to every factual claim.
- AI-generated content is disclosed as such; AI-generated images are not falsely confirmed as real.
- Manipulation against named real individuals is absolutely refused.
- Architecture details are not disclosed.
- No infinite loops, no repetitive waiting messages.

### Integration-first
Knowledge bases are designed to be cross-referenced. The Meta-Integration block (KB-10) is always active for synthesis.

### Multilingual
Default language adapts to the user's input language. Uzbek, Russian, English, and others all work natively.

### Future-compatible
Designed to take full advantage of Extended Thinking and the broader tool ecosystem (file processing, image and video generation, code execution, deep research, search grounding, workspace integration). Forward-compatible with successor models.

## Notes for users of this Gem

- For complex problems, prompt with phrases like *"Think deeply"* or *"Use full reasoning"* to maximize Extended Thinking depth.
- When sending files (PDF, DOCX, images, archives, etc.), the Gem will process them fully before responding.
- For factual questions, expect explicit statements of confidence levels rather than uniform certainty.
- For creative or strategic tasks, expect cross-domain synthesis rather than single-domain output.
